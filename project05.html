プロジェクト⑤

音声スライド生成システム

NoLangをベンチマークとした自社開発

プロジェクト概要

テキスト、PDF、Webコンテンツから高品質な動画を自動生成できるシステムを自前で開発します。ベンチマークとしてNoLangを参考にし、ビジネス資料の動画化を目的とします。

主な機能

  • 1
    テキスト・PDF・WebからのコンテンツインポートE
  • 2
    AIによるコンテンツ理解と構造化
  • 3
    スライド自動生成とレイアウト最適化
  • 4
    音声ナレーション自動生成
  • 5
    動画エンコード・エクスポート

技術スタック(案)

以下の技術スタックを提案しますが、インターンの方のご経験やご希望に応じて調整可能です。

Python React (フロントエンド) OpenAI API PDF解析ライブラリ TTS Engine FFmpeg
